# 📈 Stock Price Predictor

A Machine Learning-powered web application that predicts stock price trends using historical market data and Long Short-Term Memory (LSTM) neural networks. The application provides data visualization, trend analysis, and future stock price forecasting through an interactive Streamlit interface.

## 🚀 Features

* Historical Stock Data Analysis
* Real-Time Market Data Retrieval
* LSTM-Based Stock Price Prediction
* Interactive Data Visualization
* Moving Average Analysis
* Trend Forecasting
* User-Friendly Web Interface
* Multiple Stock Support

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YTHANISHREDDY/Stock_Price_Predictor.git
cd Stock_Price_Predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```

## 📊 How It Works

1. Fetch historical stock market data.
2. Preprocess and normalize the dataset.
3. Train an LSTM neural network.
4. Generate future stock price predictions.
5. Visualize actual vs predicted prices.
6. Analyze market trends using moving averages.
